Git e Github

   O Git é uma ferramenta de controle de versionamento de código, com o intuito de possibilitar aos desenvolvedores colaborar e desenvolver projetos de maneira eficiente.

  • Download para Windows
  • Download para Linux


    • Repositórios


      É o local em que os códigos ficam armazenados, podendo ser de maneira local ou remota em servidores.

    Comandos:

  • git init - Cria um repositório
  • git status - Verificar se existem mudanças no projeto
  • git add - Adicionar os arquivos em um projeto (um ou mais)
  • git commit -m “mensagem” - Salvar as alterações no projeto
  • git push - envia o código ao repositório remoto
  • git pull - sincroniza os dados do repositório local com o remoto
  • git clone - baixa um repositório remoto
  • git rm - remove um arquivo do git
  • git log - log das modificações no projeto
  • git reset - reseta as mudanças feitas
  • git remote rm origin - remove a origin
  • git mv - renomear/mover um arquivo para outra pasta
  • git checkout - retorna o arquivo ou rep para o estado original, sem as alterações realizadas desde o último commit
  • git remote add origin - adicionando repositório remoto ao git
  • Criar um arquivo .gitignore e colocar todos os arquivos que não queremos que vá para o repositório remoto
